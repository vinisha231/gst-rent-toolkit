# FAQ

**Where is my data stored?**
Only in your browser, using `localStorage`. It is never uploaded anywhere. Use
the Data tab to export a backup file you control.

**Will I lose my data?**
It stays until you clear your browser data or use *Data → Clear everything*. Keep
a backup (Data → Export) and you can restore it on any device.

**Is the rent inclusive of GST?**
No. Enter the rent as the taxable value; 18% GST is added on top.

**Can I use a rate other than 18%?**
Yes — each unit has its own rate field, and the calculator lets you set any rate.

**CGST+SGST or IGST?**
If the owner's GSTIN state matches the unit's place-of-supply state, it's
CGST+SGST; otherwise IGST. The toolkit picks automatically.

**Is this official / tax advice?**
No. It's a free working aid. Always confirm the final return with a qualified
accountant or GST practitioner.

# Filing GSTR-1 and GSTR-3B

For each owner (GSTIN), file two monthly returns on <https://www.gst.gov.in>.

## GSTR-1 — by the 11th of the next month

Reports each outward invoice. Rent to a GST-registered tenant is a **B2B** entry.

1. Returns Dashboard → select the month → **GSTR-1**.
2. **Table 4A (B2B)**: for each tenant, enter recipient GSTIN, invoice number and
   date, taxable value, and the rate (18%).
3. **Table 12 (HSN/SAC)**: add SAC **997212**, 18%, with the month's total.
4. Save → Submit → File with DSC/EVC.

## GSTR-3B — by the 20th of the next month

The summary return where you pay the tax.

1. Returns → select month → **GSTR-3B**.
2. **Box 3.1(a) Outward taxable supplies**: enter the month's total **taxable
   value**, plus **IGST / CGST / SGST** (use the figures on each owner card).
3. Pay the tax (cash) and file with DSC/EVC.

The dashboard's per-owner card shows exactly these numbers, and the totals update
as you change the month.

> If you opt for the **QRMP** scheme, GSTR-1/3B become quarterly with monthly tax
> via PMT-06. Confirm your scheme with your accountant.

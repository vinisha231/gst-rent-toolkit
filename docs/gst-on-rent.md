# GST on rent — the basics

Renting out **commercial / non-residential** property is a taxable supply of
service under GST, taxed at **18%**.

## Same state vs different state

- **Intra-state** (owner and property in the same state): **9% CGST + 9% SGST**.
- **Inter-state** (owner registered in a different state from the property):
  **18% IGST**.

The toolkit decides this automatically from the first two digits of the owner's
GSTIN (their state) versus the unit's **place of supply** state.

## Example (18%, intra-state)

| Rent | CGST 9% | SGST 9% | Tenant pays |
|-----:|--------:|--------:|------------:|
| 1,00,000 | 9,000 | 9,000 | 1,18,000 |

## SAC code

Commercial property rental uses **SAC 997212** — "Rental or leasing services
involving own or leased non-residential property."

## Residential rent

Renting a residence to an unregistered person is generally **exempt**. This tool
is intended for **commercial** rent. When in doubt, ask your accountant.

# Privacy

The GST Rent Toolkit is a fully client-side web app.

- **No server.** The site is static files. There is no backend to send data to.
- **No accounts, no login, no analytics, no cookies for tracking.**
- **Local only.** Your owners and units are stored in your browser's
  `localStorage` on the device you use. They are not synced or uploaded.
- **You own the data.** Export a JSON backup any time (Data tab) and import it
  elsewhere. *Clear everything* deletes it from the browser.

Because data is local to the browser, it will not appear if you switch devices or
browsers unless you import your backup file there.
